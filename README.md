# Sweets Home

## サイト概要
手作りのお菓子、そのレシピを共有できるサイト

### サイトテーマ
オリジナルのお菓子レシピのシェア

### テーマを選んだ理由
私は趣味でお菓子作りを行っています。
しかし、ネットでレシピを調べてもお菓子に特化した製作時間や製作にかかる大まかな金額が書かれていないことが多い為、
時間も金額も一目で把握できその日の状況に適したレシピを見つけやすいサイトが欲しいという思いからこのテーマを選択し
ました。



### ターゲットユーザ
趣味やバレンタイン等でお菓子を作るユーザー

### 主な利用シーン
お菓子を作りたい、作ったお菓子をほかの人にも見てもらいたいと思った時
